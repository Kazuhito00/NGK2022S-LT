# Application(Windows)
発表に使用した実行ファイルは「Windows-Build」ディレクトリに格納しています。<br>
「NGK2022S.exe」を実行してください（Windows向けビルドです）<br>
「Enter」「Page down」「テンキーの下」で次スライドに進みます。<Br>
 終了する際は「Alt」+「F4」で終了してください。
